About the Server

CarpetSMP2 is a Lifesteal Minecraft server that runs on versions 1.18 to 1.20.1, offering a cracked experience with support for both Java and Bedrock editions.

Plugins

The server uses several custom plugins including XYZ to enhance the Lifesteal experience, allow land claiming, and provide custom items.

Gameplay Mechanics

  • Steal a heart when you kill another player.
  • Lose a heart when you are killed.
  • Players can drop below 10 hearts and be eliminated permanently.
